## How this form works (READ THIS — it is an 8-step wizard)
The form has EIGHT steps shown in the left "Application progress" list:
  1. Personal Information   2. Work Eligibility   3. Experience & Skills
  4. Education   5. Resume & Cover Letter   6. Additional Questions
  7. Voluntary Self-Identification   8. Review & Submit
You advance by clicking the "Continue →" button at the bottom of each step. After you
click Continue, the page is REPLACED with the next step and EVERY element ref changes.

CRITICAL rules for getting through all 8 steps:
- ONE DROPDOWN AT A TIME: each select_option call either (a) completes the selection in one shot, or (b) opens the dropdown and tells you to click the matching option. In case (b), click the option BEFORE calling any other tool — never call select_option on a second field while a dropdown is still visually open from a previous call, or the overlay will block the second click.
- After clicking "Continue →" (or "← Back"), the refs from the previous step are GONE.
  Element refs are NOT sequential and you must NEVER guess or increment a ref
  (e.g. do NOT try e208 just because you used e207). ALWAYS re-read the live page
  snapshot and copy a ref EXACTLY as shown before acting.
- IGNORE the two decoy fields labelled "Company website" and "Home fax" — they appear on
  every step and are NOT part of the candidate's data. NEVER type anything into them.
  In particular do NOT put the LinkedIn, GitHub, or Portfolio URLs there.
- The candidate's LinkedIn / GitHub / Portfolio URLs go ONLY in the real, clearly
  LABELLED URL fields on the "Experience & Skills" step ("LinkedIn profile URL" and
  "Portfolio / GitHub URL"). Leave the LinkedIn URL for the LinkedIn field; put the
  GitHub or Portfolio URL in the "Portfolio / GitHub URL" field.
- Dropdowns (State, Country, work-authorization, etc.) are custom comboboxes: use
  select_option(target=<ref>, value="<exact option text>"), never fill/type.
- The "Earliest available start date" is a date picker: use
  select_option(target=<ref>, value="2026-07-21") — the harness navigates the calendar
  for you. Confirm the date is set before clicking Continue (the step will not advance
  while a required field is empty).
- Years-of-experience are number STEPPERS (spinbutton). Use the set_spinbutton tool:
  set_spinbutton(target=<spinbutton_ref>, value=9) — this sets the value in one call.
  The spinbutton ref is the element with role="spinbutton" (NOT the Increase/Decrease buttons).
  Do NOT use fill, select_option, or click on the spinbutton container.
- Skills are toggle buttons: click each skill button that matches the candidate's skills.
- On each step, fill EVERY required field, then click "Continue →". On the final
  "Review & Submit" step, tick the certification/consent checkboxes, enter the signature
  name, then click the Submit button and confirm the submission succeeded.
